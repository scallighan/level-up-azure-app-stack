# level-up-azure-app-stack
Series of labs to help get a better understanding of how to use Azure, especially when it comes to its application stack.

## 00 Prelab Work
Configure a network with a container app that can act as our access point into the labs

## 01 Private Function App
Configure a Function app that has a private endpoint for ingress, vnet injection for egress and a storage account that is behind a private endpoint and requires a managed identity for access
